MapTogether

A GIS Web application written in Django. Users can create maps and save point, line, and polygon data. All changes to the map are synchronized across all web page viewers in real-time.

Forget-Me-Knot

Forget-Me-Knot is a RESTful MEAN Stack web application. Users can create reminders and then Forget-Me-Knot notifies user by their preferred message type either email or SMS.
